# Vorhersage von Nebenwirkungen mit Keras – Ein einfaches neuronales Netzwerk

Dieses Projekt demonstriert eine kleine End-to-End-Machine-Learning-Pipeline mit TensorFlow/Keras, Scikit-Learn und NumPy.
Ziel ist es, eine hypothetische klinische Studie zu simulieren und ein neuronales Netzwerk zu trainieren, das vorhersagt, ob Patienten Nebenwirkungen eines neuen Medikaments basierend auf ihrem Alter erfahren.
